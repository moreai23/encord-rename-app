# encord リネームアプリ

録画エンコードファイルのファイル名を整形する Windows GUI アプリです。

---

## 起動方法

| 方法 | 手順 |
|---|---|
| **exe（推奨）** | `dist\encord_rename.exe` をダブルクリック |
| **Python 直接** | `python encord_rename_app.py` |

Python で実行する場合、標準ライブラリのみ使用するため追加インストール不要です。

---

## ファイル構成

```
rename/
├── encord_rename_app.py   ソースコード
├── dist/
│   └── encord_rename.exe  実行ファイル（Python 不要）
└── README.md              このファイル
```

設定・学習データは自動的に以下へ保存されます。

```
C:\Users\<ユーザー名>\.encord_rename_config.json
```

## 基本的な使い方（リネーム）

### 1. スキャン

1. `[...]` ボタンでフォルダを選択（前回のフォルダを自動記憶）
2. `[スキャン]` をクリック
3. 対象ファイルが一覧に表示される

### 2. プレビュー確認・編集

| 操作 | 動作 |
|---|---|
| カテゴリ列をクリック | そのファイルのカテゴリをコンボボックスで変更 |
| 変換後列をダブルクリック | ファイル名を直接手動編集 |
| `☑` 列をクリック | リネーム対象に含める / 除外する |

### 3. 複数行の一括操作

| 操作 | 動作 |
|---|---|
| `Shift` + クリック | 範囲選択（行がハイライトされる） |
| `Ctrl` + クリック | 個別に追加 / 解除 |
| ハイライト中にカテゴリ列クリック | 選択した全行のカテゴリを一括変更 |
| ハイライト中に右クリック | `☑ 対象に追加` / `☐ 対象から除外` メニュー |

### 4. 実行

`[実行]` ボタンをクリック → 確認ダイアログ → リネーム実行

## リネームルール

### スキップ対象

- `.mp4` 以外のファイル
- `!` で始まるファイル（リネーム済みと判断）

### 変換パターン

**パターン 1：タイムスタンプ形式**

```
変換前: 20260427190002102-CDTVライブ！.mp4
変換後: ! 【TV】 (2026.04.27) CDTVライブ! [1024x576 H264aac].mp4
```

**パターン 2：日付括弧形式（コーデックなし）**

```
変換前: (2026.04.29) アサデス。ＫＢＣ♡ＳｎｏｗＭａｎのメッセージ.mp4
変換後: ! 【TV】 (2026.04.29) アサデス。KBC♡SnowManのメッセージ [1024x576 H264aac].mp4
```

### 共通変換ルール

| 処理 | 内容 |
|---|---|
| 全角英数字 → 半角 | `ＫＢＣ` → `KBC`、`ＳｎｏｗＭａｎ` → `SnowMan` |
| 全角ハイフン保持 | `EIGHT－JAM` の `－` はそのまま |
| 番組表記号の除去 | `[字]` `[新]` `[再]` `[HD]` 等（設定で選択可） |
| 話数の正規化 | `＃5` `Episode6` `第２話` → `第05話`（2桁ゼロ埋め） |
| 話数ありファイル | 日付なし・カテゴリ `【アニメ】` または `【ドラマ】` |

### ファイル名フォーマット

```
話数なし: ! 【TV】 (YYYY.MM.DD) タイトル [codec].mp4
話数あり: ! 【アニメ】 タイトル 第NN話 [codec].mp4
```

---

## カテゴリ

| カテゴリ | 用途 |
|---|---|
| `【アニメ】` | アニメ（話数ありのデフォルト） |
| `【ドラマ】` | ドラマ |
| `【TV】` | 通常の TV 番組（デフォルト） |
| `【映画】` | 映画 |
| `[LIVE]` | ライブ映像 |
| `[TV]` | 音楽系 TV 番組 |
| `[MV]` | ミュージックビデオ |

カテゴリ一覧は設定ダイアログで自由に追加・変更できます。

---

## 設定ダイアログ（⚙ 設定）

| 項目 | 内容 |
|---|---|
| **プレフィックス** | 先頭の `!` をオン/オフ、文字を変更（最大5文字） |
| **番組表記号の除去** | `[字]` `[解]` `[新]` 等をチェックで除去対象に設定。`[吹]` `[字幕]` はデフォルトで保持 |
| **デフォルトカテゴリ** | スキャン時に自動設定されるカテゴリ（通常 / 話数あり） |
| **コーデック情報** | デフォルトは `[1024x576 H264aac]`（ツールバーからも変更可） |
| **カテゴリ一覧** | カンマ区切りでカテゴリを自由に追加・編集 |

> 番組表記号・カテゴリの変更はスキャン後に反映されます。  
> プレフィックス・コーデックの変更は即時プレビューに反映されます。

## アーカイブ整理（📁 アーカイブ整理）

1 ヶ月程度使用したリネーム済みファイルを、番組名ごとに整理した形式に変換します。

### 変換フォーマット

```
変換前: ! 【TV】 (2026.04.29) アサデス。KBC▽SnowManのメッセージ [1024x576 H264aac].mp4
変換後: 【TV】 アサデス。KBC (2026.04.29) SnowManのメッセージ [1024x576 H264aac].mp4
```

### 番組名・タイトルの自動分割

タイトル内の区切り文字を検出して番組名とタイトルを自動分割します。

| 優先順位 | 区切り文字 | 動作 |
|---|---|---|
| 1 | `▽` `▼` `◆` `◇` | 区切りとして削除 |
| 2 | `★` `【` `「` | タイトル側に残す |
| 3 | 全角スペース `　` | 区切りとして削除 |
| 4 | 半角スペース ` ` | 区切りとして削除（`THE` `A` `AN` は除外） |

### 編集操作

| 操作 | 動作 |
|---|---|
| 番組名列をクリック | 番組名を編集（学習済み候補がドロップダウン表示） |
| タイトル列をクリック | タイトルを編集 |
| `Shift/Ctrl` + クリックで複数選択 → 番組名列クリック | 番組名を一括変更 |
| 右クリック | `☑ 対象に追加` / `☐ 対象から除外` |
| 別のセルをクリック | 編集中の内容を確定（Enter 不要） |

### 番組名学習機能

- 番組名を入力・確定するたびに自動記録
- 次回から候補ドロップダウンに表示（最近使用順）
- **最終使用から 365 日経過で自動削除**

## exe の再ビルド

ソースコードを更新した後、以下のコマンドで exe を再生成できます。

```powershell
python -m PyInstaller --onefile --windowed --name "encord_rename" `
  --distpath "D:\claude\rename\dist" `
  --workpath "D:\claude\rename\build" `
  --specpath "D:\claude\rename" `
  "D:\claude\rename\encord_rename_app.py"
```

ビルド後は `dist\encord_rename.exe`（約 11 MB）が生成されます。  
`build\` フォルダと `encord_rename.spec` は削除しても構いません。
