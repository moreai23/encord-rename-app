# 録画エンコードファイル リネームルール

## 対象フォルダ
- パス: `Z:\` （ネットワークドライブ: `\\PAF-DESKTOP\encord`）
- paf-desktop の encord 共有フォルダ

## リネーム対象ファイル
- **日付で始まるファイルのみ**が対象
- `! [TV]`、`! [ドラマ]`、`! [アニメ]` 等のカテゴリプレフィックス付きファイルは**リネーム済みなのでそのまま**

## リネームルール

### パターン1: タイムスタンプ形式 → 日付括弧形式
- 変換前: `YYYYMMDDHHMMSS数字-タイトル.mp4`（例: `20260427190002102-CDTVライブ！.mp4`）
- 変換後: `(YYYY.MM.DD)タイトル.mp4`（例: `(2026.04.27)CDTVライブ！.mp4`）
- 先頭8桁を日付として抽出し `(YYYY.MM.DD)` 形式に変換
- 時刻コード部分（190002102 等）とハイフンは削除

### パターン2: 日付括弧形式（既存）
- 変換前: `(2026.04.29)アサデス。ＫＢＣ♡ＳｎｏｗＭａｎのメッセージ.mp4`
- 変換後: `(2026.04.29)アサデス。KBC▽SnowManのメッセージ.mp4`
- 日付フォーマットはそのまま維持

### 共通変換ルール
1. **全角英数字 → 半角に変換**（例: `ＫＢＣ` → `KBC`、`ＳｎｏｗＭａｎ` → `SnowMan`）
2. **不要な部分を削除**
3. **全角スペースの整理**

## カテゴリについて
- ファイルごとにカテゴリ（TV、ドラマ、アニメ等）が異なる
- カテゴリプレフィックス付きファイルの形式例:
  - `! [TV] (2026.04.19) タイトル [1024x576 H264aac].mp4`
  - `! [ドラマ] タイトル 第XX話 [1024x576 H264aac].mp4`
  - `! [アニメ] タイトル 第XX話 [1024x576 H264aac].mp4`

## コーデック情報
- エンコード条件: `[1024x576 H264aac]`
- この情報が付いていないファイルには追加する

## 備考
- 時刻情報は不要（日付のみ使用）
- `0102`、`0103` 等のコードは使用しない
